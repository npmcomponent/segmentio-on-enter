
# on-enter

  Attach an enter key handler.

## Installation

    $ component install segmentio/on-enter

## API

### onEnter(fn)
  Bind a handler `fn(event)` to be called when the `Enter` key is pressed.

### .unbind(fn)
  Unbinds a handler `fn`.

### .bind(fn)
  For symmetry.
